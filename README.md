## Instalación


```bash
# Clona el repositorio
git clone https://github.com/ndifabio92/Ecommerce-App-Nintendo.git

# Instala las dependencias
npm install

# Corre el server de los productos
npm run server

# Corre la aplicación
npm run dev
```
